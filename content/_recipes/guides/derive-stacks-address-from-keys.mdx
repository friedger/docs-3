# Derive a Stacks address from private and public keys

A utility function that derives a Stacks address from a <HoverLink href="hover:private-keys">private key</HoverLink> and <HoverLink href="hover:public-keys">public key</HoverLink>, with network specification (testnet/mainnet) to ensure correct address format.

## Use cases

- Message authentication and signing is one of the most common use cases for proving ownership or authenticity without exposing private keys
- Multisignature wallets when participants need to share their public keys
