# Return an entry from a map

A script that fetches and deserializes map entries from Clarity smart contracts using the Hiro API and Stacks.js, converting between Clarity values and their hexadecimal representations.

## Use cases

- Reading user balances or state from on-chain maps
- Retrieving NFT metadata stored in contract maps
- Checking governance voting records or parameters
- Fetching configuration data stored in contract maps
