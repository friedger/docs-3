# Generate a random number with block-height

<WithNotes>

A read-only function that generates a deterministic [_random_](tooltip "random") number by extracting and converting a portion of a block's <HoverLink href="hover:id-header-hash" className="text-[var(--ch-11)]">id-header-hash</HoverLink> to an unsigned integer.

## !random

This method produces deterministic pseudo-randomness, not true randomness.

Since blockchain transactions **must** be reproducible and verifiable by all nodes, true randomness isn't possible.

</WithNotes>

## Use cases

- Creating verifiable random selections for on-chain games
- Implementing fair distribution mechanisms for NFT minting
- Building lottery or raffle systems that need deterministic randomness
- Generating pseudo-random seeds for other contract operations
