## Start devnet

This endpoint starts a devnet service.

### Request Body

<Property name={"file"} type={"file"} required={true} deprecated={false}>
  
JSON file containing the devnet plan. The file can be created using `clarinet devnet package`.

<span>Example: `@default.devnet-plan.json`</span>

</Property>

### Path Parameters

<Property name={"apiKey"} type={"string"} required={true} deprecated={false}>

Hiro API key

<span>Pattern: `"0941f307fd270ace19a5bfed67fbd3bc"`</span>

</Property>

<Property name={"projectName"} type={"string"} required={true} deprecated={false}>

Project name

<span>Pattern: `"clarity-bitcoin-197s"`</span>

</Property>

| :----------- | :----------- |
