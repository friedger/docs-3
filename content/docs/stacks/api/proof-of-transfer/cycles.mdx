## Get PoX cycles

Retrieves a list of PoX cycles

### Query Parameters

<Property name={"limit"} type={"integer"} required={false} deprecated={false}>

max number of cycles to fetch

<span>Default: `20`</span>

<span>Maximum: `60`</span>

</Property>

<Property name={"offset"} type={"integer"} required={false} deprecated={false}>

index of first cycle to fetch

<span>Example: `20`</span>

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | List of cycles |
