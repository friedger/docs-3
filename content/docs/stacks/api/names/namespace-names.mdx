## Get namespace names

Retrieves a list of names within a given namespace.
### Path Parameters

<Property name={"tld"} type={"string"} required={true} deprecated={false}>

the namespace to fetch names from.

<span>Example: `"id"`</span>

</Property>

### Query Parameters

<Property name={"page"} type={"integer"} required={false} deprecated={false}>

namespace values are defaulted to page 1 with 100 results. You can query specific page results by using the 'page' query parameter.

<span>Example: `22`</span>

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | Success |
| `400` | Error |
| `404` | Error |
