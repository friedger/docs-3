## Get names owned by address

Retrieves a list of names owned by the address provided.
### Path Parameters

<Property name={"blockchain"} type={"string"} required={true} deprecated={false}>

the layer-1 blockchain for the address

<span>Example: `"bitcoin"`</span>

</Property>

<Property name={"address"} type={"string"} required={true} deprecated={false}>

the address to lookup

<span>Example: `"1QJQxDas5JhdiXhEbNS14iNjr8auFT96GP"`</span>

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | Success |
| `404` | Error |
