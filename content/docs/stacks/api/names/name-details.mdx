## Get name details

Retrieves details of a given name including the `address`, `status` and last transaction id - `last_txid`.
### Path Parameters

<Property name={"name"} type={"string"} required={true} deprecated={false}>

fully-qualified name

<span>Example: `"muneeb.id"`</span>

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | Success |
| `400` | Error |
| `404` | Error |
