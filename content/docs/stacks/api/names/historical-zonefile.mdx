## Get historical zone file

Retrieves the historical zonefile specified by the username and zone hash.
### Path Parameters

<Property name={"name"} type={"string"} required={true} deprecated={false}>

fully-qualified name

<span>Example: `"muneeb.id"`</span>

</Property>

<Property name={"zoneFileHash"} type={"string"} required={true} deprecated={false}>

zone file hash

<span>Example: `"b100a68235244b012854a95f9114695679002af9"`</span>

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | Success |
| `400` | Error |
| `404` | Error |
