## Call read-only function

Call a read-only public function on a given smart contract.

The smart contract and function are specified using the URL path. The arguments and the simulated tx-sender are supplied via the POST body in the following JSON format:


### Request Body

map of arguments and the simulated tx-sender where sender is either a Contract identifier or a normal Stacks address, and arguments is an array of hex serialized Clarity values.

<Property name={"sender"} type={"string"} required={true} deprecated={undefined}>

The simulated tx-sender

</Property>

<Property name={"arguments"} type={"array<string>"} required={true} deprecated={undefined}>

An array of hex serialized Clarity values

</Property>
### Path Parameters

<Property name={"contract_address"} type={"string"} required={true} deprecated={false}>

Stacks address

<span>Example: `"SP3D6PV2ACBPEKYJTCMH7HEN02KP87QSP8KTEH335"`</span>

</Property>

<Property name={"contract_name"} type={"string"} required={true} deprecated={false}>

Contract name

<span>Example: `"mega"`</span>

</Property>

<Property name={"function_name"} type={"string"} required={true} deprecated={false}>

Function name

<span>Example: `"get-balance"`</span>

</Property>

### Query Parameters

<Property name={"tip"} type={"string"} required={false} deprecated={false}>

The Stacks chain tip to query from

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | Success |
