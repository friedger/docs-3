Call a read-only public function on a given smart contract.

The smart contract and function are specified using the URL path. The arguments and the simulated tx-sender are supplied via the POST body in the following JSON format:

### Request body

Map of arguments and the simulated tx-sender where sender is either a contract identifier or a normal Stacks address, and arguments is an array of hex serialized Clarity values.

<Property required={true} deprecated={false} name={"sender"} type={"string"}>

The simulated tx-sender

</Property>

<Property required={true} deprecated={false} name={"arguments"} type={"array of string"}>

An array of hex serialized Clarity values

</Property>

### Path parameters

<Property required={true} deprecated={false} name={"contract_address"} type={"string"}>

Stacks address

<span>Example:  `"SP187Y7NRSG3T9Z9WTSWNEN3XRV1YSJWS81C7JKV7"`</span>

</Property>

<Property required={true} deprecated={false} name={"contract_name"} type={"string"}>

Contract name

<span>Example:  `"imaginary-friends-zebras"`</span>

</Property>

<Property required={true} deprecated={false} name={"function_name"} type={"string"}>

Function name

<span>Example:  `"get-token-uri"`</span>

</Property>

### Query parameters

<Property required={false} deprecated={false} name={"tip"} type={"string"}>

The Stacks chain tip to query from

</Property>

| Status code | Description |
| :----------- | :----------- |
| `200` | Success |
