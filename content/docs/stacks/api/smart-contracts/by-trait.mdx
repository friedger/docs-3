## Get contracts by trait

Retrieves a list of contracts based on the following traits listed in JSON format -  functions, variables, maps, fungible tokens and non-fungible tokens
### Query Parameters

<Property name={"trait_abi"} type={"string"} required={true} deprecated={false}>

JSON abi of the trait.

</Property>

<Property name={"limit"} type={"integer"} required={false} deprecated={false}>

max number of contracts fetch

</Property>

<Property name={"offset"} type={"integer"} required={false} deprecated={false}>

index of first contract event to fetch

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | List of contracts implement given trait |
