## Get the value of a constant inside a contract

Attempt to fetch the value of a constant inside a contract. The contract is identified with [Stacks Address] and [Contract Name] in the URL path. The constant is identified with [Constant Name].

In the response, `data` is the hex serialization of the constant value.

### Path Parameters

<Property name={"contract_address"} type={"string"} required={true} deprecated={false}>

Stacks address

</Property>

<Property name={"contract_name"} type={"string"} required={true} deprecated={false}>

Contract name

</Property>

<Property name={"constant_name"} type={"string"} required={true} deprecated={false}>

Constant name

</Property>

### Query Parameters

<Property name={"tip"} type={"string"} required={false} deprecated={false}>

The Stacks chain tip to query from. If tip == latest, the query will be run from the latest known tip (includes unconfirmed state).

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success |
| `400` | Failed to retrieve constant value from contract |
