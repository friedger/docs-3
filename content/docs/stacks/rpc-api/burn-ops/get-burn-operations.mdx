## Get burn operations

Get all burn operations of type `op_type` successfully read at `burn_height`. Valid `op_type`s are `peg_in`, `peg_out_request` and `peg_out_fulfill`.

### Path Parameters

<Property name={"burn_height"} type={"integer"} required={true} deprecated={false}>

height of the burnchain (Bitcoin)

</Property>

<Property name={"op_type"} type={"string"} required={true} deprecated={false}>

name of the burnchain operation type

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Burn operations list |
