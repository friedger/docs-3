## Fetch a Nakamoto block

Fetch a Nakamoto block by its index block hash.

### Path Parameters

<Property name={"block_id"} type={"string"} required={true} deprecated={false}>

The block's ID hash

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | The raw SIP-003-encoded block will be returned. |
| `404` | The block could not be found |
