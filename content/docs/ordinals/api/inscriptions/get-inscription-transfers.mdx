Retrieves all transfers for a single inscription.

### Query parameters

<Property required={false} deprecated={false} name={"offset"} type={"integer"}>

Result offset

<span>Minimum: `0`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

Results per page

<span>Minimum: `1`</span>

<span>Maximum: `60`</span>

</Property>

### Path parameters

<Property required={true} deprecated={false} name={"id"} type={"Any properties in string, integer"}>

Inscription unique identifier (number or ID)

</Property>

| Status code | Description      |
| :---------- | :--------------- |
| `200`       | Default response |
| `404`       | Default response |

</div>
