toc: false
---
Clone the [Ordhook repository](https://github.com/hirosystems/ordhook) to your local machine and install via `cargo`.

```terminal
$ git clone https://github.com/hirosystems/ordhook.git
$ cd ordhook
$ cargo ordhook-install
```
