## Inscription transfers

Retrieves all transfers for a single inscription

### Query Parameters

<Property name={"offset"} type={"Offset"} required={false} deprecated={false}>

Result offset

<span>Minimum: `0`</span>

</Property>

<Property name={"limit"} type={"Limit"} required={false} deprecated={false}>

Results per page

<span>Minimum: `1`</span>

<span>Maximum: `60`</span>

</Property>

### Path Parameters

<Property name={"id"} type={"Inscription Identifier"} required={true} deprecated={false}>

Inscription unique identifier (number or ID)

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | Default Response |
| `404` | Default Response |
