## Transfers per block

Retrieves a list of inscription transfers that ocurred at a specific Bitcoin block

### Query Parameters

<Property name={"block"} type={"Block Identifier"} required={true} deprecated={false}>

Bitcoin block identifier (height or hash)

</Property>

<Property name={"offset"} type={"Offset"} required={false} deprecated={false}>

Result offset

<span>Minimum: `0`</span>

</Property>

<Property name={"limit"} type={"Limit"} required={false} deprecated={false}>

Results per page

<span>Minimum: `1`</span>

<span>Maximum: `60`</span>

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | Default Response |
| `404` | Default Response |
