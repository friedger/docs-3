## BRC-20 Token Holders

Retrieves a list of holders and their balances for a BRC-20 token

### Query Parameters

<Property name={"offset"} type={"Offset"} required={false} deprecated={false}>

<span>Minimum: `0`</span>

</Property>

<Property name={"limit"} type={"Limit"} required={false} deprecated={false}>

Results per page

<span>Minimum: `1`</span>

<span>Maximum: `60`</span>

</Property>

### Path Parameters

<Property name={"ticker"} type={"string"} required={true} deprecated={false}>

</Property>
