## Satoshi Inscriptions

Retrieves all inscriptions associated with a single satoshi

### Query Parameters

<Property name={"offset"} type={"Offset"} required={false} deprecated={false}>

Result offset

<span>Minimum: `0`</span>

</Property>

<Property name={"limit"} type={"Limit"} required={false} deprecated={false}>

Results per page

<span>Minimum: `1`</span>

<span>Maximum: `60`</span>

</Property>

### Path Parameters

<Property name={"ordinal"} type={"Ordinal Number"} required={true} deprecated={false}>

Ordinal number that uniquely identifies a satoshi

<span>Minimum: `0`</span>

</Property>

| Status code | Description |
| :------ | :------ |
| `200` | Default Response |
| `400` | Default Response |
